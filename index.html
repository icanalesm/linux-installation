Overview

This is a personal guide for installing a Linux-based system in UEFI mode with full disk encryption. Encryption is done using LUKS on the partitions (see the ArchWiki).

The system has the following characteristics:

  • Boot loader: rEFInd
  • Network manager: NetworkManager
  • Firewall: firewalld
  • Time synchronization: chrony
  • Power management: TLP
  • Display server: Xorg
  • Sound server: PipeWire
