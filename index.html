Overview

This is a personal guide for installing a Linux-based system in UEFI mode with full disk encryption.

Encryption will be done using LUKS on the partitions (see the ArchWiki). The following partition scheme will be used:

PartitionType
BootEFI
Encrypted systemext4
Encrypted homeext4

The system has the following characteristics:

  • Boot loader: rEFInd
  • Network manager: NetworkManager
  • Firewall: firewalld
  • Time synchronization: chrony
  • Power management: TLP
  • Display server: Xorg
  • Sound server: PipeWire
