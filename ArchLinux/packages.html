Arch Linux - Packages for minimal installation

During installation, install the groups base and base-devel.

During configuration, remove the following packages (from group base):

  • dhcpcd
  • jfsutils
  • lvm2
  • mdadm
  • nano
  • netctl
  • reiserfsprogs
  • s-nail
  • xfsprogs
  • vi

During configuration, install the following packages:

  • git
  • openssh
  • unzip
  • vim
  • wget
