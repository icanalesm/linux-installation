rEFInd - Configuration

Get rEFInd themes

$ git clone https://github.com/bobafetthotmail/refind-theme-regular.git
$ git clone https://github.com/htower/refind-theme-regular-black.git
$ sudo mkdir -p /boot/efi/EFI/refind/refind-theme-regular/icons
$ sudo cp refind-theme-regular/icons/256-96/* /boot/efi/EFI/refind/refind-theme-regular/icons
$ sudo cp refind-theme-regular-black/icons/256-96/bg.png /boot/efi/EFI/refind/refind-theme-regular/icons/bg_black.png
$ sudo cp refind-theme-regular-black/icons/256-96/selection-big.png /boot/efi/EFI/refind/refind-theme-regular/icons/selection_black-big.png
$ sudo cp refind-theme-regular-black/icons/256-96/selection-small.png /boot/efi/EFI/refind/refind-theme-regular/icons/selection_black-small.png

Comment all configuration values in /boot/efi/EFI/refind/refind.conf and specify the configuration file to include

/boot/efi/EFI/refind/refind.conf
...
include myconf.conf
...

Create the configuration file /boot/efi/EFI/refind/myconf.conf

/boot/efi/EFI/refind/myconf.conf
#
# myconf.conf
# Configuration file for the rEFInd boot menu
#

timeout 20
use_nvram false
hideui hints
use_graphics_for linux,osx,windows
showtools shell, gdisk, memtest, mok_tool, apple_recovery, windows_recovery, hidden_tags, firmware, reboot, exit

icons_dir refind-theme-regular/icons
big_icon_size 256
small_icon_size 96
banner refind-theme-regular/icons/bg_black.png
selection_big refind-theme-regular/icons/selection_black-big.png
selection_small refind-theme-regular/icons/selection_black-small.png
