openSUSE Leap 15.0 - Packages for minimal installation

During installation, install the following packages:

  • aaa_base
  • dosfstools
  • grub2-x86_64-efi
  • kernel-default
  • kexec-tools
  • mokutil
  • shim

During configuration, install the following packages:

  • ca-certificates
  • ca-certificates-mozilla
  • gcc
  • git
  • iproute2
  • less
  • logrotate
  • make
  • man
  • man-pages
  • pciutils
  • pcmciautils
  • psmisc
  • tar
  • unzip
  • unzip-doc
  • usbutils
  • vim-data
  • wget
  • yast2
